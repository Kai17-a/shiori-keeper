# Frontend

このディレクトリは Nuxt 4 の SPA を含む。

## 起動

```bash
bun install
bun run dev
```

## テスト

```bash
bun run test
bun run e2e
bun run e2e:run
bun run e2e:headed
```

## 主要ファイル

- [Nuxt 設定](./nuxt.config.ts)
- [アプリ本体](./app/)
- [Playwright 設定](./playwright.config.ts)
- [Vitest 設定](./vitest.config.ts)
- [テスト](./tests/)
