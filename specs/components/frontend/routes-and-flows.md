# ルートとフロー

## ルート

| Route           | Purpose                                            |
| --------------- | -------------------------------------------------- |
| `/`             | ダッシュボードの概要と最新ブックマーク             |
| `/bookmarks`    | ブックマーク一覧、作成、編集、削除、検索           |
| `/favorites`    | お気に入りブックマーク一覧                         |
| `/folders`      | フォルダ一覧と作成                                 |
| `/folders/[id]` | フォルダ詳細、配下ブックマーク一覧、名前変更、削除 |
| `/tags`         | タグ一覧と作成                                     |
| `/tags/[id]`    | タグ詳細、関連ブックマーク一覧、名前変更、削除     |
| `/rss`          | RSS フィード一覧、作成、編集、削除、実行            |
| `/rss/[id]`     | RSS フィード詳細、記事一覧                          |
| `/settings`     | テーマ切り替え                                     |

## ユーザーフロー

### ダッシュボード

- 集計値と最新ブックマークのプレビューを表示する
- ブックマーク、フォルダ、タグ、RSS、お気に入りの画面へ遷移できる
- サイドバーの子要素を描画する前にカタログデータを読み込む

### ブックマーク

- URL、タイトル、任意の説明、フォルダ、タグ付きでブックマークを作成する
- モーダルでブックマークを編集する
- 確認後にブックマークを削除する
- タイトルまたは URL で検索する
- フォルダとタグで絞り込む
- カードまたはリンクから外部 URL を開く
- お気に入り一覧ではスター状態の切り替え、編集、削除を行える

### フォルダ

- 一覧ページからフォルダを作成する
- モーダルでフォルダ名を編集する
- フォルダ詳細ページを開く
- 確認後にフォルダを削除する
- 選択フォルダに属するブックマークを表示する
- フォルダ詳細から関連ブックマークの編集、削除、お気に入り切り替えができる

### タグ

- 一覧ページからタグを作成する
- モーダルでタグ名を編集する
- タグ詳細ページを開く
- 確認後にタグを削除する
- 選択タグに関連するブックマークを表示する
- タグ詳細から関連ブックマークの編集、削除、お気に入り切り替えができる

### RSS

- 画面上部で Discord webhook 設定の読込、疎通確認、保存を行う
- RSS 定期実行の有効/無効を切り替える
- 一覧ページから RSS フィードを作成する
- モーダルで RSS フィードを編集する
- RSS フィード詳細ページを開く
- 確認後に RSS フィードを削除する
- RSS フィードを手動実行できる
- RSS フィード詳細では取得済み記事一覧をページング付きで表示する

### 設定

- テーマ切り替えを行える

## 共通操作

- 主要なナビゲーションはサイドバーを使う
- CRUD 操作はカードとモーダルを使う
- フォルダ/タグ/RSS フィード更新後はサイドバー用データを再取得して同期を保つ
