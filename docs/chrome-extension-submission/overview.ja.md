# 概要

## 短文

Shiori Keeper の Chrome 拡張機能です。現在開いているページのタイトルとURLを Shiori Keeper API サーバーに保存し、フォルダやタグを付けて整理できます。API サーバーの接続確認にも対応しています。

## 長文

Shiori Keeper は、ブラウザで開いているページをそのままブックマークとして保存できる Chrome 拡張機能です。現在のページのタイトルとURLを取得し、ローカルで動作する Shiori Keeper の API サーバーへ登録します。

保存時には、あらかじめアプリ側で管理しているフォルダやタグを選択でき、ブックマークを後から探しやすい形で整理できます。また、拡張機能から API サーバーへの接続確認も行えるため、保存先の動作状態を確認しながら利用できます。

拡張機能は API サーバー URL を `chrome.storage.local` に保存し、既存ブックマークがある場合は URL をもとに更新します。該当するブックマークがない場合は新規作成にフォールバックします。

この拡張機能は Shiori Keeper 本体と組み合わせて使うことを前提としており、保存されたデータは Shiori Keeper のデータベースで管理されます。

プロジェクトページ: https://github.com/Kai17-a/shiori-keeper
